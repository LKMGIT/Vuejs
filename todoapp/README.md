### 요구사항 정리

1. 사용자가 할 일을 입력하고 만약 입력이 비어있지 않은 상태로 enter or click 진행시 LocalStorage 에 값 저장 

2. TodoList는 LocalStorage가 비어있지 않으면 리스트에 스토리지 안에 있는 값을 추가

3. 사용자는 checkbox를 사용하여 완료 상태를 변경할 수 있고 삭제 버튼을 눌러 해당하는 item 만 삭제를 할 수 있다. 

4. 또한 사용자는 전체 삭제 버튼을 이용하여 List의 값 전체를 삭제 할 수 있다. 

5. 완료/미완료 필터로 구분하여 검색 가능
